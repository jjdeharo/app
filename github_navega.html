Navegador de repositorios de GitHub

Navega por repositorios y convierte URLs de GitHub.

v3.4

Consejo: GitHub limita las consultas anónimas. Añadir un token personal aumenta este límite, evitando bloqueos temporales al navegar.

Cómo obtener un Token de Acceso Personal
  1. Ve a la página de creación de tokens de GitHub.
  2. En "Note", pon un nombre, ej: HerramientaRepos.
  3. En "Select scopes", marca **SOLAMENTE** la casilla `repo`.
  4. Haz clic en "Generate token". Cópialo y pégalo a continuación.
