Navegador de repositorios de GitHub

Navega por repositorios y convierte URLs de GitHub.

v4.0

Cómo obtener un Token de Acceso Personal

GitHub limita las consultas anónimas. Añadir un token personal aumenta este límite, evitando bloqueos temporales, y te permite ver tus repositorios privados.

  1. Ve a la página de creación de tokens de GitHub.
  2. En "Note", pon un nombre, ej: HerramientaRepos.
  3. En "Select scopes", marca **SOLAMENTE** la casilla `repo`.
  4. Haz clic en "Generate token". Cópialo y pégalo a continuación.

Ayuda

¿Qué hace esta aplicación?

Esta herramienta te ayuda a interactuar con repositorios de GitHub de dos maneras principales, accesibles a través de las pestañas superiores.

Pestaña: Convertir URL

Esta es la función más directa. Te permite obtener rápidamente diferentes versiones de una URL de GitHub.

  • Uso: Pega cualquier URL de GitHub (un repositorio, un archivo, etc.) y haz clic en "Convertir".
  • Resultados: La aplicación te mostrará enlaces directos a la URL del repositorio, al archivo en formato "raw" y, si está disponible, a su versión en GitHub Pages.
  • Examinar Carpeta: Junto a los resultados, verás un botón que te llevará directamente a la pestaña "Navegar" para ver el contenido de la carpeta donde se encuentra el archivo que has convertido.

Pestaña: Navegar

Esta sección te permite explorar el contenido de los repositorios de GitHub como si fuera un explorador de archivos.

  • Sin Token: Puedes escribir el nombre de cualquier usuario de GitHub para ver sus repositorios públicos y navegar por ellos.
  • Con Token: Si introduces un Token de Acceso Personal de GitHub, la aplicación se conectará a tu cuenta. Se mostrará tu nombre de usuario y tendrás botones para ver "Mis Repositorios" y "Mis Gists". También aparecerá un botón de "Explorar Usuario Público" que te permitirá buscar en otros perfiles sin desconectarte.
  • Navegación: Haz clic en un repositorio para ver su contenido. Si entras en una carpeta, la ruta aparecerá en la parte superior (migas de pan), permitiéndote volver atrás fácilmente.
